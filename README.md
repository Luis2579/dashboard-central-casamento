# Dashboard Central de Casamento

Dashboard financeiro e operacional para planejamento de casamento, com módulos integrados de fornecedores, pagamentos, convidados, RSVP, mesas, checklist, cronograma, contratos, buffet, decoração, música e presentes.

## Execução local

```powershell
python server.py
```

Acesse `http://127.0.0.1:4173`.
